## Секция готовых домов
- Не использовать маленький надзаголовок `Готовые решения`.
- Заголовок секции выровнен как отдельный полноширинный блок: `Дома для жизни за городом без хаоса на стройке`.
- Блок домов реализован как слайдер из 5 карточек, по 2 карточки на десктопе и по 1 на мобильных.
- Данные домов находятся в массиве `homes` в `script.js`.
- В галерее фото сортируются так: сначала наружные фото дома, затем внутренние фото, затем планировки.
- На десктопе карточки слайдера ограничены по высоте, чтобы в первый экран помещался весь блок, а не только фотографии.
- Правая панель модального окна прокручивается отдельно.
- Для `Ломоносовское подворье (зеленый)` первая фотография в карточке и галерее должна быть:
  `фотографии домов для слайдера/Ломоносовское подворье (зеленый)/photo_2026-05-20 00.01.58.jpeg`
- Для `Ломоносовское подворье (рыжий)` первая фотография в карточке и галерее должна быть:
  `фотографии домов для слайдера/Ломоносовское подворье (рыжий)/photo_2026-05-20 00.20.58.jpeg`
- В `script.js` порядок наружных фото задается через `exteriorImagesByHome`; не менять сортировку на простую алфавитную, иначе обложки и порядок галереи собьются.
- Источник фото: папка `фотографии домов для слайдера`, внутри 5 папок:
  - `Аннинские просторы`
  - `Иннола парк`
  - `КП Инноволо`
  - `Ломоносовское подворье (зеленый)`
  - `Ломоносовское подворье (рыжий)`
- По клику на карточку открывается модальное окно-галерея: большое фото, стрелки, миниатюры, закрытие по фону, кнопке и `Esc`.
